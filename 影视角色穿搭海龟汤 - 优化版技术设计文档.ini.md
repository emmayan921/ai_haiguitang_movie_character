影视角色穿搭海龟汤 - 优化版技术设计文档
（对齐 Agent 指令 | 语义 AI 匹配 | 前后端分离）
1. 项目概述
基于 React + TypeScript + Vite + Tailwind CSS 开发的影视角色穿搭主题 AI 海龟汤游戏。
流程：首页（国内 / 国外分类）→ 难度选择（入门 / 进阶 / 挑战）→ 游戏提问 → 汤底揭晓。
采用 AI 语义理解 判断用户问题，返回「是 / 否 / 无关」；6 次提问未猜中自动弹出穿搭关键提示；通过 Node.js+Express 后端 代理智谱 AI API，解决跨域并保护密钥安全。
2. 技术栈（与训练营一致）
前端：React 18 + TypeScript 5 + Vite 5 + Tailwind CSS + React Router 6
状态：React Hooks (useState/useEffect/useRef)
后端：Node.js + Express + TypeScript
AI：智谱 AI GLM-4 原始 API（后端代理调用）
部署：Vercel（前端静态站点 + 后端 Serverless）

4. 核心类型定义（TypeScript）
typescript
运行
// 地区类型
type TRegion = 'domestic' | 'international'

// 难度类型
type TDifficulty = 'starter' | 'advanced' | 'challenge'

// 单题故事结构（无关键词匹配）
interface TStory {
  id: string
  region: TRegion
  difficulty: TDifficulty
  title: string
  soupSurface: string       // 汤面线索
  canonicalAnswer: string    // 正确角色名
  sourceWork: string        // 所属影视
  outfitAnalysis: string    // 穿搭解析
  imageUrl: string          // 角色配图
  keyHint: string           // 6次提问后的提示
}

// 对话消息
interface TMessage {
  id: string
  role: 'user' | 'assistant'
  content: string
  createdAt: number
}
5. 页面与路由

/reveal/:storyId → 汤底揭晓页
6. 核心逻辑（AI 语义匹配版）
用户进入题目，查看汤面
用户输入任意问题
前端请求后端接口 /api/ai/ask
后端通过智谱 AI 语义理解判断问题
严格返回：是 / 否 / 无关
每提问一次计数 + 1
满 6 次未猜对 → 自动弹出 keyHint 关键提示
可随时查看汤底，进入揭晓页
7. UI 风格（固定不变）
全局背景：红色帷幕静态图
装饰：静态汤婆婆 AI 二创图（不遮挡交互）
卡片：白底半透明、圆角、阴影
主色：红色系
聊天气泡：左右区分，自动滚动
8. 后端接口说明
POST /api/ai/ask
入参：question、storyId
逻辑：根据题目信息做语义判断
出参：{answer: "是" | "否" | "无关"}
安全：API Key 仅存在后端环境变量
9. 规则约束（必须实现）
不使用关键词匹配（qaPairs 已删除）
AI 只允许返回三个字：是 / 否 / 无关
6 次提问自动解锁提示
题库放在前端，便于维护
前后端分离部署，无跨域问题
