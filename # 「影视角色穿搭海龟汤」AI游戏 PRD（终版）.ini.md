# 「影视角色穿搭海龟汤」AI游戏 PRD（终版）
## 一、产品概述
### 1.1 产品名称
核心名称：影视角色穿搭海龟汤（副标题：经典角色穿搭大盘点）
- 备注：名称聚焦「穿搭」核心（贴合自媒体定位），简洁易记，适配游戏玩法与粉丝记忆习惯。

### 1.2 核心定位
一款以「经典影视角色穿搭特征」为核心线索的AI海龟汤轻量网页游戏，后续可适配微信小程序（挂载视频号）；玩家先选择「国内/国外影视角色」分类，再选择难度层级，通过向AI提Yes/No问题推理角色身份，揭晓汤底时同步展示角色穿搭解析；页面UI融入AI二创的汤婆婆元素（宫崎骏风格），打造差异化视觉风格，服务于「拆解影视角色穿搭」自媒体账号，主打「互动娱乐+穿搭内容延伸」，无需下载、点击即玩。

### 1.3 核心目标
- 短期：训练营练手，做出可运行的网页版游戏，给自媒体粉丝互动娱乐；
- 长期：适配视频号小程序，提升账号互动率，后续通过广告/带货变现。

## 二、目标用户（聚焦核心）
1. 核心用户：自媒体账号粉丝（18-35岁，关注影视角色穿搭、喜欢追剧的女性为主）；
2. 泛用户：轻推理爱好者（通勤/摸鱼碎片化娱乐，接受5分钟一局的小游戏）；
3. 潜在用户：对影视穿搭感兴趣的路人（通过游戏引流关注自媒体账号）。

## 三、核心功能
### 3.1 MVP（必须做，小白优先落地）
#### 3.1.1 游戏大厅首页（核心层级：首页→分类选择→难度选择）
- 第一层：首页展示「国内影视角色穿搭」「国外影视角色穿搭」两大核心分类卡片；
  - 视觉展示：分类卡片标注「分类名+代表角色穿搭缩略图（AI原创）」，背景融入AI二创汤婆婆轻量插画元素（如汤婆婆轮廓装饰）；
  - 核心按钮：「进入分类」（卡片内）、「游戏规则」（顶部右侧）；
- 第二层：点击分类后，展示该分类下的「入门级/进阶级/挑战级」难度卡片；
  - 视觉展示：难度卡片标注「难度+角色数量+代表穿搭配图」；
  - 核心按钮：「开始游戏」（难度卡片内）、「返回首页」（顶部固定）；
- 适配要求：卡片尺寸适配手机/电脑，按钮点击区域≥44px（提前适配小程序规范），汤婆婆元素不遮挡核心操作区域。

#### 3.1.2 海龟汤游戏页面
- 顶部区域：显示「当前分类+难度+题目序号」（如“国外影视 - 入门级 - 第1题”）+ 轻量规则提示（“靠穿搭猜角色，仅可提Yes/No问题，6次未猜对将显示穿搭关键提示～”）；
- 核心区域：
  1. 汤面展示：居中显示「纯穿搭线索」（如“红色科技感战甲，胸口有发光反应堆，富豪人设穿搭”），展示区背景融入AI二创汤婆婆汤锅极简图案；
  2. 聊天交互：底部固定输入框（提示示例：“可提问：是漫威角色吗？穿搭是裙装吗？”）→ 输入后AI仅回复“是/否/无关” → 对话历史按「你：XXX / AI：XXX」分行展示，支持滚动查看；
- 规则触发：连续6次提问未推理出角色，自动弹出「穿搭关键提示」（如“提示：该穿搭是漫威钢铁侠的经典红金战甲，核心特征是反应堆装饰”）；
- 核心按钮：「查看汤底」（6次提示后解锁）、「返回难度选择页」、「结束游戏」（返回首页）；
- 交互细节：输入框支持手机输入法，按钮点击有“轻微变色”反馈，汤婆婆元素仅作为装饰，不影响操作。

#### 3.1.3 汤底页面
- 核心展示（按优先级排序）：
  1. 角色穿搭原创配图：居中显示（尺寸300px×400px），自适应页面，点击可放大/缩小，加载失败时显示“穿搭图片加载中～”占位符；
  2. 核心信息：角色名 + 所属影视 + 核心穿搭标签（如“钢铁侠（托尼·斯塔克）-《复仇者联盟》- 红金科技感战甲穿搭”）；
  3. 穿搭解析：1-2句简短解析（如“该穿搭核心是红金配色+反应堆装饰，体现科技感与富豪人设的结合”）；
  4. 引流引导：小字提示“想拆解更多影视角色穿搭，关注我的视频号👉【你的账号名】”；
- 视觉装饰：页面角落融入AI二创汤婆婆极简插画（如汤锅图案），营造“揭晓汤底”的仪式感；
- 核心按钮：「再来一局」（同分类同难度下一题）、「返回难度选择页」、「返回首页」。

### 3.2 核心功能补充
#### 3.2.1 影视角色穿搭题库规范
每道题必须包含3部分，按以下格式整理（方便后续写代码调用）：
| 字段         | 国内影视示例                          | 国外影视示例                          |
|--------------|---------------------------------------|---------------------------------------|
| 汤面（穿搭线索） | 清宫旗装，旗头配点翠装饰，素雅色系    | 红色科技感战甲，胸口有发光反应堆      |
| 核心线索（Yes/No答案） | 是清宫剧角色？→是；穿搭是男装？→否    | 是漫威角色？→是；穿搭是裙装？→否      |
| 汤底         | 甄嬛-《甄嬛传》- 清宫旗装穿搭 - 配图链接 | 钢铁侠-《复仇者联盟》- 红金战甲穿搭 - 配图链接 |
| 穿搭解析     | 旗装核心是素雅色系+点翠旗头，体现角色温婉人设 | 红金配色+反应堆装饰，体现科技感与富豪人设 |
- 分级设计（按分类拆分）：
  1. 国内影视分类：
     - 入门级：全民熟知角色（甄嬛、刘培强、白娘子）；
     - 进阶级：经典配角（《琅琊榜》梅长苏、《武林外传》佟湘玉）；
     - 挑战级：冷门配角（经典港片复古穿搭配角、老国产剧小众角色）；
  2. 国外影视分类：
     - 入门级：全民熟知角色（钢铁侠、哈利波特、奥黛丽赫本）；
     - 进阶级：经典配角（《哈利波特》斯内普、《泰坦尼克号》露丝）；
     - 挑战级：冷门配角（老欧美片复古风衣配角、小众海外剧角色）。

#### 3.2.2 配图实现规则
- 汤婆婆UI配图：使用AI二创后的汤婆婆图片（仅作装饰元素，无商用属性），保存为PNG透明格式，上传至免费图床（imgchr.com），复制链接嵌入代码；
- 角色穿搭配图：仅使用AI原创穿搭图/免费可商用素材（Pexels/Mixkit），禁止使用影视原截图/官方海报（避免侵权）；
- 展示逻辑：汤婆婆元素仅作为页面装饰（角落/背景），不遮挡核心内容，角色穿搭图加载失败显示占位符，不影响游戏流程。

### 3.3 后续可做（MVP跑通后再加）
1. 本地收藏：答错/想回顾的角色加入“我的收藏”（纯前端localStorage实现，无需后端）；
2. 用户登录：微信小程序版接入微信登录（后续变现需要）；
3. 游戏记录：仅做本地记录，暂不做跨用户排行榜（避免后端复杂度）；
4. 自定义汤底/多人对战：粉丝量破万后再评估（现阶段无必要）。

## 四、界面设计要求（保留汤婆婆元素+小白可落地）
### 4.1 整体风格
- 核心风格：宫崎骏汤婆婆风格+轻复古影视氛围感；
- 主色调：浅米色+复古棕+少量酒红（呼应红帷幕/汤锅的温暖感，适配女性用户）；
- 核心视觉元素：AI二创汤婆婆图片（极简轮廓/汤锅图案），仅作为装饰，不做复杂交互；
- 适配要求：符合微信小程序视觉规范，无夸张配色，按钮/输入框适配手机操作。

### 4.2 各页面细节（小白能实现的汤婆婆UI）
#### 4.2.1 游戏大厅首页
- 布局：「国内/国外」分类卡片居中，汤婆婆二创图片作为页面右侧/底部静态装饰（尺寸180px×200px，半透明）；
- 视觉：分类卡片背景用“复古电影胶片纹理”，卡片hover/点击时轻微上移（CSS实现），汤婆婆元素不遮挡卡片内容。

#### 4.2.2 游戏页面
- 布局：聊天界面模仿微信对话框（CSS实现），汤面展示区背景嵌入汤婆婆汤锅极简图案（低透明度），汤婆婆人物插画放在页面左下角（静态，尺寸150px×180px）；
- 视觉：汤面文字居中，对话历史区清晰，汤婆婆元素仅作装饰，不影响输入/查看对话。

#### 4.2.3 汤底页面
- 视觉：页面角落放置汤婆婆二创插画（汤锅图案），揭晓汤底时角色穿搭图渐入显示（CSS opacity动画），保留“揭晓真相”的仪式感；
- 排版：配图居中，穿搭解析在配图下方，引流引导文字用浅红色突出，汤婆婆元素不抢焦点。

## 五、技术栈建议（小白友好）
### 5.1 第一阶段（网页版，训练营练手）
- 前端：纯 HTML + CSS + JavaScript（不用任何框架，训练营基础完全够用）；
- AI交互：Coze内置AI（替代DeepSeek/Claude API，无需申请APIKey、无需写后端）；
- 配图存储：免费图床（imgchr.com），汤婆婆二创图/角色穿搭图均上传至此，复制链接嵌入代码；
- 部署：Vercel（免费，一键部署，生成网页链接）。

### 5.2 第二阶段（小程序版，视频号挂载）
- 开发方式：微信开发者工具 + 低代码模板（复用网页版核心代码）；
- AI交互：微信小程序接入Coze小程序插件（无缝复用现有逻辑）；
- 部署：微信小程序平台（需完成个体认证，300元/年）。

## 六、合规&版权要求（关键）
1. 汤婆婆元素合规：使用AI二创后的汤婆婆图片，仅作为非商用页面装饰，不提取角色商业价值，规避版权风险；
2. 角色穿搭配图合规：仅使用AI原创穿搭图/免费可商用素材，禁止使用影视原截图、官方海报、角色肖像；
3. 文字合规：仅描述“穿搭特征+影视类型”，不引用独家台词/剧情，小众版权角色用“穿搭标签+影视类型”代替全名（如“清宫旗装穿搭 - 国产清宫剧”）；
4. 交互合规：无诱导分享（不强制分享才能玩下一题），小程序版接入微信防沉迷插件（官方现成）。
